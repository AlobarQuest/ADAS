# README for AI Helpers (web-app Starter)

Welcome to an ADAS-enabled `web-app` project.

You must:

1. Read `.ai/00_ai-entry-point-meta-rules.md`.  
2. Determine **Usage Mode** (Light vs Heavy).  
3. Load ADAS as defined in:
   - `docs/for-ai-helpers-how-to-load-adas-context.md`
   - `docs/adas-usage-modes.md`
4. Respect:
   - Core Domains + Mode-Aware Addenda
   - web-app Profile Domains
   - Project `.local` overlays

Light Mode → small, local changes.  
Heavy Mode → full governance.

Do not weaken security, architecture, or testing rules under any circumstances.
